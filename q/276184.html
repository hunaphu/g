Method

The value of most French numbers in \$[0\dots100]\$ is the sum of the values of all parts, e.g. soixante-dix-sept is \$60+10+7=77\$.

The only exception is for quatre-vingts and all numbers of the form quatre-vingt-xxx where quatre-vingt(s) has to be interpreted as \$4\times20\$ rather than \$4+20\$. We can handle this special case by setting the value of quatre to \$60\$ instead of \$4\$ if it appears at the first position and is followed by other words.

Lookup table

The modulo chain and the corresponding lookup table were found by running a random brute force search and are unlikely to be optimal.
