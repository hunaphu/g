lambda n:(c:=n&-n)%3==1>~n//c%8

Try it online!

This uses Legendre's three-square theorem, that \$n\$ is a sum of 3 squares unless it has the form \$n=4^a(8b+7)\$. The output is True for such n, False otherwise.

We first extract c, the largest power-of-2 divisor in n, using the classic bit trick n&-n. Check that this c is \$1 \bmod 4\$, which indicates that c is a power of 4, the \$4^a\$ above. This also short-circuit eliminates n==0 which would divide-by-0 in the next part.

Next, we check that n//c%8==7. This is equivalent to ~(n//c)%8<1, and it turns out ~n//c%8<1 works too.
