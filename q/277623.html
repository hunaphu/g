A function taking an argument in the form [12,00] and returning a newline separated string.

It forms a canvas of lines of 40 characters including the newline. In the link these are periods but can be replaced by spaces by editing the period in the first line. It then draws from the tip of the minute hand to the tip of the hour hand.

The centre of the clock is at index 420 (I didn't plan it that way, it just happened!) There is an array of 6 magic numbers for the offsets corresponding to half a revolution of the hand. the other half revolution is managed by negating these offsets.

let M = [...Array(15)].map(_ => [..."".padEnd(32)]);

for(let n = 0; n < 12; n++) {
  [ 0, 1.072, 2, 2.9 ].forEach(i => {
    let q = 0.524 * n - 1.57,
        y = 7.5 + Math.sin(q) * i * 7 / 3 | 0,
        x = 15.5 + Math.cos(q) * i / 3 * (n % 6 - 3 ? 496 % ~n % 2 + 13 : 15.1) | 0;

    M[y][x] = i ? n >> 1 : "#";
    M[y][x + 1] = i ? n * 5 % 10 : "#";
  });
}
