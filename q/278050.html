Method

We generate all 43,046,721 4x4 grids filled with F, O, X as flattened strings.

For each grid, we build the string consisting of a leading zero, followed by the flattened grid, followed by the separator string "[object Object]", followed by another zero, followed by the reversed flattened grid.

For instance, the grid FFFFFXXXXXOOOOOO is turned into:

0FFFFFXXXXXOOOOOO[object Object]0OOOOOOXXXXXFFFFF

For a grid to be valid, none of the following regular expression parts should give a match on this string:

  1. (F|X)(\w*\1){5} → more than 5 F or 5 X
  2. (O\w*){7} → more than 6 O
  3. 0(....)*.?F(O|....O....)X → horizontal or diagonal FOX
  4. 0(....)*.?..F..O..X → anti-diagonal FOX
  5. F...O...X → vertical FOX

Note that parts 3 and 4 are actually factorized as:

0(....)*.?(F(O|....O....)|..F..O..)X
