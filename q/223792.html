Try it online!

  • Separator inside a group is (space)
  • Separator between groups is - (space and minus sign)
  • If the array start with a 1, it prepends a separator

how it work:

  • `-n` is the string representation of -n (it is 2 character long)

With strings:

  • if 2 consecutive number are equal : n-p = 0 => "-n"[0:] = "-n"
  • if the absolute difference is equal to 1: n-p = 1 or -1 => "-n"[1:] = "-n"[-1:] = "n"
  • if the difference is greater than 1 (say for example 2): n-p = 2 => "-n"[2:] = "" but ""or-n = -n
  • if the difference is lesser than -1 (say for example -2): n-p = -2 => "-n"[-2:] = "-n"
