a.map(p =       // start with p non-numeric
(v, i) =>       // for each value v at index i in a[]:
  v |           //   if v is not zero
  a.find(v =>   //   or the first non-zero value in a[]
    i-- * v < 0 //   at an index higher than i (*)
  ) != p ?      //   is not equal to p:
    p = v       //     set p to v
  :             //   else:
    p           //     use the current value of p
)               // end of map()

(*) The expression i-- * v < 0 is true if and only if i is negative and v is non-zero.
