Named function that gets the longest common subsequence between two strings. Name the function _ and call it with arguments x and y as in =_(A1, A2). Returns one of the subsequences of max length, or blank when there is no common subsequence. Note that subsequence is not the same as substring. This is NP-complete because David Maier (1978).

Ungolfed version as regular formula:

        a, lcs(lcs, x, head(y)),
