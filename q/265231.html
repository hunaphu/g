f=(n,m=1)=>n*m%1?f(n,m+1):n*m

Try it online!

Recursive approach, pretty straightforward. I have no doubt that there is a better solution using some more clever tricks.

Code golf solution ending explained

f=(n,m=1)=>n*m%1?f(n,m+1):n*m  // full program

f=                             // define f
  (n,                          // taking input n
     m=1)                      // and an *optional* input m defaulting to 1
         =>                    // and outputting
           n*m%1?              // if n*m is an integer multiple of the input n
                         :n*m  //   that very same integer multiple of n
                               // else
                 f(n,m+1)      //   the result of calling f with m increased by 1
                               // (so that n*m is now the *next* multiple of n)
