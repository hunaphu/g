Admittedly, the image uses fiftythree shades of grey instead of three colors.
And the ASCII art could be golfed a bit further, but who's counting? 640 bytes ought to be enough for anybody.

The only section where some actual code golfing could take place is

$g|% C*o $o
$g|% D*e
[Text.Encoding]::UTF8|% G*g(,($o|% T*y))

which ungolfs to

$g | ForEach-Object -MemberName CopyTo $o
$g | ForEach-Object -MemberName Dispose
[Text.Encoding]::UTF8 | ForEach-Object -MemberName GetString (,($o | ForEach-Object -MemberName ToArray))

which is a more complicated version of

$g.CopyTo($o)
$g.Dispose()
[Text.Encoding]::ASCII.GetString($o.ToArray())
