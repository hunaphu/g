    xor edi, edi    # Set EDI to 0 by XORing it with itself.

How?

ḅ‘Ṁ - Link: list of integers, D (N in base ten) e.g. [  3,   2,   0,   2,   1]
 ‘  - increment each digit of {D}                    [  4,   3,   1,   3,   2]
